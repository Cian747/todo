# Todo App

A lightweight task management application designed to help users organize, track, and complete their daily activities with ease.

## Features

- Create, edit, and delete tasks
- Mark tasks as completed
- Organize tasks with priorities or categories
- Simple and intuitive user interface
- Lightweight and easy to run locally

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Cian747/todo.git
   cd todo
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the application:

   ```bash
   npm start
   ```

## Usage

- Add a new task by entering the task details and selecting save.
- Edit an existing task to update the title, description, or due date.
- Mark tasks as completed to keep track of progress.
- Remove tasks that are no longer needed.
